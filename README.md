"Приложение для поиска команды или пары "
[ ПРОБЛЕМНОЕ ПОЛЕ ]
Многим учащимся хотелось бы участвовать в различных событиях, связанных с учебным заведением, олимпиадной активностью и личной жизнью, однако найти команду или пару для участия в любом из мероприятий- не всегда легкая задача. Учащиеся школ, лицеев и университетов сталкиваются с трудностями в поиске команды, партнеров или же просто друзей.
[ПОТЕНЦИАЛЬНАЯ АУДИТОРИЯ ]
Продукт может заинтересовать широкую аудиторию, однако имеет настройки в виде представления учебного заведения и группы/класса для отдельных групп пользователей. К таким группам относятся:
Учащиеся средних/ старших классов
Участники различных групповых олимпиад/хакатонов
Создатели групповых/парных проектов
Адаптирующиеся к новому учебному заведению
Участники массовых школьных/ университетских мероприятий, гдде нужна пара или команда

[ПОХОЖИЕ ПРОДУКТЫ]
1. Авито￼
Один из крупнейших сервисов объявлений в России.
Функционал:
публикация объявлений;
поиск по категориям и регионам;
встроенный чат между пользователями;
фильтрация объявлений;
возможность прикреплять фото и описание.

2. Работа.ру￼
Платформа для поиска исполнителей и заказчиков.
Функционал:
создание задач и откликов;
личные сообщения;
рейтинги и отзывы;
фильтрация по специализации.

3. Discord￼
Популярная платформа для общения и тематических сообществ.
Функционал:
текстовые и голосовые каналы;
тематические серверы;
личные сообщения;
групповые обсуждения.
 Причины, по которым продукт подходит для поиска лучше мессенджеров:
1.Посты быстро теряются
В группах и чатах объявления быстро уходят вниз из-за постоянного потока сообщений. Пользователю приходится регулярно публиковать сообщение заново
2.Мессенджеры неудобны для организации поиска:
перегруженности чатов;
большого количества спама;
отсутствия структуры сообщений;
сложности поиска старых диалогов;
смешивания личного общения и учебных контактов.
3.Отсутствие специализированного поиска по учебным целям
